# VIS

Visualization of rendering pipeline

# Build

Requirements:
- CMake version 3.28
- C++ compiler

## Windows & Linux

~~~
$ git clone --recursive https://github.com/ICEPINK/VIS.git
$ cd VIS/
$ cmake -B ./build/ -DCMAKE_BUILD_TYPE=Release
$ cd build/
$ cmake --build
~~~
